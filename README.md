Automatizando Infraestrutura com AWS CloudFormation (DIO)

Este repositório foi desenvolvido para o desafio da DIO chamado “Automatizando Infraestrutura com AWS CloudFormation”. O projeto apresenta uma infraestrutura automatizada utilizando templates CloudFormation, com documentação detalhada e exemplos práticos para demonstrar o domínio de Infraestrutura como Código (IaC).

Objetivos do Projeto

Implementar uma infraestrutura automatizada na AWS usando CloudFormation.

Criar templates reutilizáveis para componentes essenciais de uma aplicação.

Demonstrar boas práticas de modularização e documentação em IaC.

Publicar o repositório no GitHub como portfólio técnico.
